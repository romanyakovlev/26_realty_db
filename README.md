# Cinemas Site

Site that shows apartments sorted by region.

# What in use
- Flask
- WTForms
- SQLAlchemy

# Installing

clone repo:
```sh
git clone https://github.com/romanyakovlev/24_telegraph
```

create database file and fill it with data:
```sh
python3 create_tables_and_fill_db.py
```

run app locally:
```sh
python3 server.py
```
